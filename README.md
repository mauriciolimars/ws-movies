# WS Movies API
## REST API for movie producer award interval analysis.

## Requirements 
### - Java 21+
### - Maven 3.6+

# Run Application 
 
## ```mvn spring-boot:run```

# Run Tests
## ```mvn test```

# API
## Endpoint ```GET /ws-movies/producers/awards-intervals```


# Testing 
## ```curl http://localhost:8080/ws-movies/producers/awards-intervals```
